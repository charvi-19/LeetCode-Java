# LeetCode 27 - Remove Element

## Problem
Remove all occurrences of a given value in-place and return the number of remaining elements.

LeetCode Link:
https://leetcode.com/problems/remove-element/

## Approach
- Use two pointers.
- Traverse the array using one pointer.
- Whenever an element is not equal to `val`, copy it to the next available position.
- Return the count of remaining elements.

## Time Complexity
O(n)

## Space Complexity
